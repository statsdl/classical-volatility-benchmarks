# Classical Volatility Benchmarks

A GitHub-ready benchmark suite for classical volatility forecasting, realized GARCH-based modeling, risk evaluation, visualization, statistical testing, and sensitivity analysis.

The repository includes RV, RBV, RTV, TrueSD, HAR-RV, GARCH-family models, realized GARCH-based models, VaR, ES, CRPS, coverage probability, statistical tests, sensitivity analysis, and GitHub-rendered result reports.
